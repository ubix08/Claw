# Tool Notes

## Built-in Tools

The following tools are always available:
- `read` — read files and images
- `bash` — execute shell commands
- `edit` — replace exact text in files
- `write` — write/create files

## Skills

Skills extend these tools with knowledge about specific CLIs and APIs.
Check `workspace/skills/` for installed skills.

## Conventions

- Always read a skill's SKILL.md before using it for the first time
- For file edits: prefer `edit` (surgical) over `write` (overwrite)
- For bash: use timeout flags on long-running commands
